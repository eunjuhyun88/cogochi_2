# 2026-03-18 Steam Release Reframe

## Goal

Reframe Cogochi into a Steam-ready product flow where character selection, growth planning, short proof battles, and visible improvement are stronger than raw systems density.

## Scope

- evaluate the existing game direction against a releaseable player journey
- preserve chart battle, training, and rental fantasies while fixing the front door
- define the new main loop and screen responsibilities
- align the core source-of-truth docs to the new flow
